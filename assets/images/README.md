# Images Directory

This directory contains images and assets for the Point Zero project.

## Contents

- **logo.png** - Project logo (to be added)
- Other project images

## Recommended Formats

- PNG for logos with transparency
- JPG for photos
- SVG for vector graphics

## Usage

Images are referenced in HTML/CSS using relative paths:
```html
<img src="assets/images/logo.png" alt="Point Zero Logo">
```

```css
background-image: url('../assets/images/logo.png');
```
